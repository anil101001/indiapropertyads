# 🚀 Quick Start Guide

## Get Started in 3 Steps

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Run the Application
```bash
npm run dev
```

### Step 3: Open in Browser
Navigate to: **http://localhost:3000**
